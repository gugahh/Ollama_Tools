# Ollama_Tools
Python Tools based on Ollama for IA based helpers

## Ollama Translate
- Python application for translating SRT subtitles in any language to Brazilian Portuguese;
- Works very well with local gemma:e4b, but will accept other models.
- Running locally you won't hit your token limit!
- Takes around 3 hours to translate a subtitle in my machine.
- Shows estimated and ellapsed time indicators, and previews the translated text.
- Command line only, no GUI.
